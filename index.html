What is SVF?

SVF is an interprocedural Static Value-Flow analysis on top of LLVM. It allows value-flow construction and pointer analysis to be performed in an iterative way, thereby providing an effective solution for both.

What kind of analyses does SVF provide?

  • Andersen's Pointer analysis and call graph construction
  • Sparse Flow-sensitive Pointer Analysis
  • Value-Flow dependence analysis
  • Interprocedural memory SSA
  • Detecting source-sink related bugs, such as memory leak, incorrect file-open close error.
